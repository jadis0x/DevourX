# DevourX
DevourX is a powerful mod menu developed for the Devour. Written in C++ and independent of any injectors (using version proxy), it provides players with full control and enhanced gameplay options.


> [!WARNING]
> I have decided to make the DevourX project open-source. Feel free to modify it as you wish. However, I will only provide assistance to those who have donated to me before. Please do not send messages asking how to install it.


## Key Features of DevourX
- **High-performance C++ codebase:** DevourX is built using C++, ensuring optimal performance and smooth execution.
- **Seamless integration with Devour:** No external injectors are required; DevourX integrates directly into the game for a hassle-free experience.
- **User-friendly interface:** Designed for ease of use, the intuitive menu system allows players to quickly access powerful features.

## Features
- Unlimited UV light
- Big flashlight
- Faster walking speed
- Instant Win
- Rank Spoof
- Flying Mode
- Spawn any objects or carry
- Force lobby start
- EXP Modifier
- Unlock All
- No long interact
- Kill, revive, or send jumpscares to players

And much more!
